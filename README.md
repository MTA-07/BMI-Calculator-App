# BMI Calculator App 🧮

A simple, user-friendly Body Mass Index (BMI) calculator built with Python and Tkinter.

## Features ✨
* **Graphical User Interface (GUI):** A clean and custom layout built using Tkinter.
* **Error Handling:** Implemented `try-except` blocks to prevent application crashes when invalid data (like letters) is entered. Shows intuitive error pop-ups using `messagebox`.
* **Dynamic Results:** Automatically calculates the BMI and categorizes the user as Underweight, Normal Weight, Overweight, or Obese with color-coded text outputs.

## Prerequisites 🛠️
You need to have Python installed on your system to run this application. Tkinter comes pre-installed with standard Python distributions.
